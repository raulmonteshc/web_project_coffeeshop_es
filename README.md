# ☕ Triple Espresso

Landing page estática para una cafetería ficticia llamada **Triple Espresso**, desarrollada con HTML5 y CSS.  
El proyecto presenta una página moderna con secciones informativas, formulario de reserva y diseño limpio enfocado en la experiencia visual.

---

## 📌 Descripción

**Triple Espresso** es un proyecto frontend creado para practicar maquetación web y estilos con CSS puro.  
Simula el sitio web oficial de una cafetería, incluyendo información sobre horarios, ubicación y reservas.
